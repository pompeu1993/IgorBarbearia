# Skill Marketplace

Use this skill as an index for local workspace skills.

## When To Invoke

- The user asks what skills exist
- A task could benefit from one of several skills
- New skills were added and need organization
- You want to recommend the best skill before execution

## Responsibilities

- List available skills
- Summarize what each skill does
- Explain when each skill should be invoked
- Suggest missing skills for recurring workflows

## Expected Output

Provide a short catalog with:

1. Skill name
2. Purpose
3. Trigger condition
4. Typical outputs
